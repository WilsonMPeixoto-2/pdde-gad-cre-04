# Registro de Regras Operacionais Locais

Este documento relaciona orientações da 4ª CRE/GAD que dependem de fonte formal, confirmação institucional ou validação operacional antes de serem apresentadas como obrigações definitivas no Guia PDDE no SEI!RIO.

## Regra de publicação

Uma orientação local somente poderá ser publicada como obrigatória quando possuir:

- fonte formal identificada;
- unidade emissora;
- exercício ou período de aplicabilidade;
- data de verificação;
- responsável pela revisão;
- redação aprovada para publicação.

Enquanto esses elementos não estiverem registrados, a orientação deverá permanecer com status `pending-local-validation` e não poderá ser apresentada como regra definitiva.
