# Registro de Revisão Normativa

Este documento controla a revisão humana das orientações normativas, procedimentais e operacionais publicadas no Guia PDDE no SEI!RIO.

A aprovação técnica do pipeline não substitui a revisão normativa humana nem a homologação das rotinas locais.

## Campos obrigatórios

| Campo | Preenchimento |
|---|---|
| Identificador da regra | ID utilizado na camada normativa |
| Título | Nome público da orientação |
| Fonte oficial | Norma, ato, manual, comunicado ou orientação formal |
| Dispositivo ou seção | Artigo, inciso, item, página ou seção aplicável |
| Aplicabilidade | Entidade, exercício, ação e categoria econômica |
| Revisor | Nome da pessoa que realizou a conferência |
| Função/área | Cargo, unidade ou competência do revisor |
| Data | Data da revisão |
| Resultado | Aprovado, ajustado ou pendente |
| Observações | Limitações, condicionantes e providências futuras |

## Regra de encerramento

Uma linha somente poderá receber o resultado **Aprovado** depois que a fonte, o dispositivo, a aplicabilidade e a redação exibida no site tiverem sido conferidos por pessoa com competência material sobre o tema.
