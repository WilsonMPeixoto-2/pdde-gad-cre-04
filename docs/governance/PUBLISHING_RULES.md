# Regras de Publicação Normativa

Este arquivo registra as regras mínimas para publicação de orientações normativas, procedimentais ou operacionais no guia PDDE no SEI!RIO.

## Regra obrigatória

> Nenhuma orientação sobre prazo, documento obrigatório, vedação, forma de pagamento, sistema federal, contratação, tributação, autenticação, nível de acesso ou fluxo decisório poderá ser publicada sem fonte oficial identificada, delimitação de aplicabilidade e data de verificação.

## Aplicação imediata

- Regras federais devem indicar fonte oficial do FNDE ou norma federal aplicável.
- Procedimentos municipais devem indicar ato, manual, decreto, guia institucional ou orientação formal da Prefeitura/SME/SEI!RIO.
- Orientações locais da CRE/GAD sem documento formal devem ser tratadas como pendentes de validação local.
- Auditoria automatizada não substitui revisão normativa humana.
- Conteúdo com status pendente não deve ser apresentado como obrigação.
